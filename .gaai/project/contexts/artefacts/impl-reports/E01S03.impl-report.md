# Implementation Report: E01S03 — Watch config files and hot-reload on change

## Summary

Added fsnotify-based config file watching and hot-reload to the Clincus web server. The system watches `/etc/clincus/config.toml` and `~/.config/clincus/config.toml`, debounces changes, and reloads config thread-safely. If the dashboard port changes, the HTTP listener is gracefully cycled.
